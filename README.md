# Chess Engine

This is a fun little Chess Engine I made in C++. It compiles with CMake, and uses C++ 17. 
There are no dependencies to this project. 

## Installation

To install this repository, simply copy it:

```powershell
git clone https://github.com/The-codeMachine/ChessEngine.git
```

## Compilation & Running the Chess Engine

To compile and run the chess engine simply build the cache, and then the program:

```powershell
# Cache construction

mkdir build
cd build

cmake ..
cd ..

# Builds the program
cmake --build build
```
